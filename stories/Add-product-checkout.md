As a product owner I would like the user to be able to select a product add it to the cart and checkout
## Acceptance criteria

- User should be able to click add button on the product .
- User can verify the product is added in Cart 
- Verify the Quantity and price
- User should be able to checkout by pressing Checkout button
- User should be able to see the order confirmation
- User shoubd be able to dismiss the order confirmation by clicking continue shopping
