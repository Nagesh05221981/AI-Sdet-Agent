As a product owner I would like the user to be able to select one  product, add it to the cart and checkout
## Acceptance criteria

- User should be able to click add button on any one  product .
- User can verify the same product is added in Cart 
- Verify the Quantity and price of the product added to cart
- User should be able to checkout by pressing Checkout button
- User should be able to see the order confirmation
- User shoubd be able to dismiss the order confirmation by clicking continue shopping
- User should be able to add more than 1 product to cart and checkout
- User should be able to remove a product from cart.
- User should be able to increase/decrease quantity of product alreadi in cart
