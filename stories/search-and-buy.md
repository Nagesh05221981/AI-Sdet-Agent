As a product owner I would like the user to be able to search one  product, add it to the cart and checkout
## Acceptance criteria

- User should be able to search a product (headphone) .
- User should be able to see the successfull search result(headphone) 
- User should be able to add the product in search result to cart
- User should be able to checkout by pressing Checkout button
- User should be able to see the order confirmation
- User shoubd be able to dismiss the order confirmation by clicking continue shopping
- User should be able to clear the search bar , delete the product (headphone) from searchbar
